Code for Loan-Approval-Prediction competition on kaggle classification problem):
I/ EDA and Preprocess
  1/ Import and clean data.
  2/ Feature engineering, label encoder, feature importance
  
II/ MODELING: (random forest, catboost, XGBoost, LGBM clasifier)
  1/ Over sampling using SMOTE
  2/ Hyperparameters tuning using Optuna
  3/ Evaluate using metric
  
III/ Ensemble use Hill climbing algorithm
